b:orderingList

The b:orderingList is used to sort a list.

Basic Usage:

Select items, then move them up or down using the button controls. Alternatively use the drag points to move item.

Select multiple items with the ctrl key, or click-drag the mouse over multiple items.

Table Layout:

Use a columnar layout to achieve more rich control over your item display.

Nested ajax behavior:

Nest an ajax tag to trigger server-side actions for client-side events.

Drag to select:

Allow selection of items by dragging, then sort the items by dragging by the corner.

Containment and no buttons:

Prevent dragging outside of the list and hide the buttons.

Disable mouse:

With mouse disabled the items can only be sorted by the buttons.
