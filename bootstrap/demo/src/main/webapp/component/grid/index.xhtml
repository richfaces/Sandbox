b:gridContainer b:gridRow b:gridColumn

The Bootstrap grid is a responsive layout that adapts to the width of your screen. Its composed by three components :

  • b:gridContainer is a wrapper around your page / content.
  • b:gridRow is a section of your page and will occupied full width availabled.
  • b:gridColumn is a part of a gridRow. There is maximum 12 gridColumns in one gridRow.
