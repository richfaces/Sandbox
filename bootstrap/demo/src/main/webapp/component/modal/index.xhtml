b:modal

The b:modal is used to display a modal panel in top of the current page.

There is two ways to use the modal component : the basic one and the custom one. Learn in the two following examples.

Basic usage

The basic usage of the modal suppose that your modal will be divided in 3 parts :

  1. the header : only containing the title of the modal, a h3 title by default. Can be set using the header attribute or the header facet.
  2. the body : it will be composed of the children of your modal component.
  3. the footer : should contains buttons or links in order to allow the user to interact with your modal. Can be set using the footer facet.

Each part will be surrounded by a div tag with a specific CSS class (modal-header, modal-body, modal-footer). Those div will be generated for you.

Custom usage

If the default structure doesn't correspond to your needs, you will probably to customize with no restriction. You can do that of course. Just turn the custom attribute to true and that's it! No more auto-generated div, you can put whatever you want inside your modal componant.

It's safe to say that even if you want to customize freely the content of your modal, you might still need the div with specific CSS classes. In order to no bother you by learning them, you can use 3 corresponding JSF components:

  1. modalHeader
  2. modalBody
  3. modalFooter
