All Bootstrap power

Do you think Bootstrap, from Twitter, is awesome? If yes, so we do. If not, we still do (sorry). All the awesomeness of Bootstrap, all its great components, all its effects, all has been (or will be) migrated as JSF components! It's nearly magic and we love it.

Beautiful Font-Awesome icons

Want icons on your applications? Want them beautiful? Want them scalable? Discover Font Awesome icons, a totally free collection of amazing icons. And guess what? They are Bootstrap compatible and we wrapped them in JSF too!

Community project

RichFaces Bootstrap is not a false open-source project with read-only code, it's a true community project, so you can participate! Fork the GitHub repo, fill issues on our JIRA, join weekly meetings or just come to say hi on #richfaces IRC chan at irc.freenode.net.

As awesome as Bootstrap is, it's never enough... no never! So we are gonna add some new smooth features to it but also use jQuery UI plugins or write our own code and pimp that with Bootstrap design in order to extend the number of RichFaces Bootstrap components.

Living on the bleeding edge

HTML5, CSS3, ECMAScript5, LESS... did you ear about those (new) technologies? They have started to change the face of the web and RichFaces Bootstrap supports them, providing you new components, new attributes and new effects.

Sandbox project

Just a little warning about the fact that this project is still a sandbox project under heavy development so tag and attribute names can change at any time. Use it for fun or prototype purpose only for now. And stay tuned for the release of course!

Big demos

Those demos are a copy/paste of original Twitter Bootstrap demos but using JSF components instead of HTML tags. They will allow you to quickly browse over all RichFaces Bootstrap features and see nice examples. They are situated on the left of the top navbar.

Full documentation and VDL

Each RichFaces Bootstrap component has a dedicated documentation page which will provide you all informations you need to know in order to use it. The VDL will also be present on those pages. You can browse them by using the top navbar, components have been grouped by theme.
