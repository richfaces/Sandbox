This component is based on Agile carousel, a jQuery plugin. Note that this component can work in 2 ways. You can hard code the slides :

<carousel:carousel> <h:graphicImage value="img/banner_bike.jpg"/> <h:graphicImage value="img/banner_paint.jpg"/> </carousel:carousel>

or you can iterate over some collection or datamodel (presence of value attribute decides):

<carousel:carousel value="\#{carouselBean.data}" var="img"> <h:graphicImage value="\#{img.photo}"/> \#{img.title} </carousel:carousel>
